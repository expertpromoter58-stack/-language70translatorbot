# 🌍 @language70translatorbot

A Telegram bot that translates text between 70+ languages with automatic language detection.

## Features

- 🌐 70+ Languages support
- 🔍 Auto-detects source language
- 🎯 Custom target language selection
- 📱 Interactive buttons
- ⚡ Fast translations

## Commands

- `/start` - Welcome message
- `/help` - Help information
- `/settings` - View settings
- `/languages` - List all languages
- `/language` - Change target language

## Deployment

This bot is deployed on Railway.

### Environment Variables

- `TELEGRAM_TOKEN`: Your bot token from @BotFather
