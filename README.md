# 🌍 Language70 Translator Bot

A powerful Telegram bot that translates text between 70+ languages with automatic language detection.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/deploy)

## ✨ Features

- 🌐 **70+ Languages** - Support for 70+ languages worldwide
- 🔍 **Auto-Detection** - Automatically detects source language
- 🎯 **Custom Target** - Set your preferred target language
- 📱 **User-Friendly** - Simple commands and interactive buttons
- ⚡ **Fast Translation** - Powered by Google Translate API
- 🔒 **Privacy-Focused** - No message history stored

## 🚀 Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and get welcome message |
| `/help` | Show help information |
| `/about` | About this bot |
| `/settings` | View your current settings |
| `/languages` | List all supported languages |
| `/language` | Change target translation language |
